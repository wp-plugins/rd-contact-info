== Description ==

Provides a way to add dynamic contact information that includes more than just the admin email. Includes shortcodes and theme functions. For complete usage instructions see: http://www.reitzdesigns.com/2010/06/rd-contact-info-plugin/

== Installation ==

1. Upload the `rd_contact_info` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu.
3. To add your contact infomation go to `Edit Contact Information` the under `Settings` menu.
4. For complete instructions and examples of shortcodes and themes, click on `visit plugin page` under 'RD Contact Info' on the `Plugins` page in wordpress.

== Frequently Asked Questions ==

= What if I want to add contact information not provided in the form, such as Skype or MSN or etc? =
There is a work around for this. If there is an entry your not using enter the information in that field. 
For example you could enter your Skype ID in the 'Moble' field then use this shortcode:
[rd-contact-mobile label="Skype ID:"]
